# MR/ Dampella M.V - Official School Website

A modern, responsive, and feature-rich school website for MR/ Dampella M.V, Southern Province, Sri Lanka. Includes a comprehensive LMS (Learning Management System) portal for students and teachers.

## 🚀 Features

- **Dynamic Homepage**: Modern design with interactive hero section, stats, and school overview.
- **Admin Panel**: Secure dashboard for managing posts, events, messages, and valid student IDs.
- **LMS Portal**: 
  - **Student Dashboard**: View marks, download past papers, and access school resources.
  - **Teacher Dashboard**: Manage student marks, assignments, and class communications.
- **Supabase Integration**: Real-time database and authentication.
- **Internationalization**: Full support for both English and Sinhala languages.
- **Theme Support**: Dark and light mode with system default detection.

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Database/Auth**: Supabase
- **Icons**: Lucide React
- **Themes**: Next Themes
- **Internationalization**: Custom i18n context

## 📦 Getting Started

### Prerequisites

- Node.js && pnpm
- Supabase account

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dilshanmethsara/dampellamv.git
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Set up environment variables:
   Create a `.env.local` file with your Supabase credentials.

4. Run the development server:
   ```bash
   pnpm dev
   ```
